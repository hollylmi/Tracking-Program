How to prepare your SVG from AutoCAD

The easiest workflow is to upload your AutoCAD file directly as a DXF. Plytrack automatically detects all enclosed panel regions and assigns IDs — no manual work needed.

Draw in AutoCAD (no special prep)
  • Draw panel boundaries using any combination of lines, polylines, or arcs. They can overlap and share edges — Plytrack finds the enclosed regions automatically.
  • Optionally add a TEXT label inside each panel (e.g. P-001) so panels are named automatically. If omitted, panels get numbered panel_1, panel_2… and you can rename them in the app.
Export from AutoCAD
  1. Go to File → Save As (or SAVEAS command).
  2. In the Files of type dropdown choose AutoCAD 2010 DXF (*.dxf).
  3. Save and upload the .dxf file into a layer below using the Convert DXF button.
Optional: add a background image

If your drawing is based on a site scan or PDF, you can upload that as a background image (JPG, PNG, or PDF) so the diagram shows the real site underneath the panel outlines.

Manual SVG option

Advanced users can also upload a hand-crafted Plain SVG where each panel is a <polygon> or <path> with a unique id attribute.
