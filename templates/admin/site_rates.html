How It Works
  1. Add all equipment and resources that have an hourly cost when on site (cranes, generators, traffic control, etc.)
  2. On each daily entry, select which items are on site that day and set the quantity
  3. When a delay is recorded, the delay cost is automatically calculated using these rates × delay hours
  4. This replaces the old system of using machine group/individual delay rates from the equipment page
